# 🎯 A-6 Digital Tools Buying Website



Welcome to the **A-6 Digital Tools Buying Website**! This is a simple and user-friendly platform where users can browse various digital tools, view pricing plans, and manage their shopping cart in real-time.



---



## 📖 Project Description

This project is an interactive e-commerce style web application. It allows users to toggle between a product gallery and a shopping cart. It features a modern design based on Figma, high-quality UI components, and real-time alerts when adding or removing items.



## 🛠️ Technology Used

* **React.js** - For building the user interface.

* **Tailwind CSS & DaisyUI** - For styling and responsive design.

* **JavaScript (ES6+)** - For logic and functionality.

* **React-Toastify** - For professional popup notifications.

* **JSON** - For storing and managing product data.



## 🌟 Top 3 Features

1. **Interactive Cart Toggling:** Switch easily between viewing the product list and your selected items using the center toggle buttons.

2. **Real-time Cart Updates:** The Navbar shows exactly how many items you have selected. When you click "Buy Now," the cart count increases instantly!

3. **Smart Notifications:** Uses `React-Toastify` to give you instant feedback whenever you add an item, remove one, or complete your checkout.
